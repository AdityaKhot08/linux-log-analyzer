Linux Log Analyzer

Automated log analysis and report generator using GitHub Actions (CI). This project demonstrates Linux commands, shell scripting, Git, CI/CD and deployment to GitHub Pages.

How it works

  1. analyze.sh parses system logs and creates report.txt
  2. GitHub Actions runs the script on a schedule
  3. Report is uploaded and published to GitHub Pages

Files in this repo

  • analyze.sh - main script
  • .github/workflows/report.yml - CI workflow
  • index.html - project page
  • Dockerfile (optional) - static-site container
  • README.md - this guide
